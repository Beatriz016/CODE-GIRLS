----------- Acesso total ao S3 e EC2
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Action": [
                "s3:*",
                "ec2:*"
            ],
            "Resource": "*"
        }
    ]
}

----------- Acesso de leitura ao S3 e EC2

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Action": [
                "ec2:Describe*",
                "s3:Get*",
                "s3:List*"
            ],
            "Resource": "*"
        }
    ]
}

EC2: A ação "ec2:Describe*" permite todas as operações de descrição para os recursos do EC2.
S3: As ações "s3:Get*" e "s3:List*" permitem operações de leitura e listagem em buckets e objetos do S3.
